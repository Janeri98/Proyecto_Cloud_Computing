## ❓ Preguntas Frecuentes

### ¿Puedo editar el sitio web?
**Sí, claro.** Edita `index.html`, `styles.css` o `script.js` y sube los cambios a Azure.

### ¿Cuánto tiempo toma?
**2-3 horas** si sigues la guía paso a paso. Quizás más si no tienes experiencia con Azure.

### ¿Es difícil?
**No.** La guía es muy clara y fácil de seguir. Sin necesidad de código complejo.

### ¿Qué pasa si me equivoco?
**No hay problema.** Puedes eliminar todo y empezar de nuevo. No pierdes dinero.

### ¿Necesito pagar?
**No.** Con Azure for Students obtienes $200 en crédito gratuito. Este proyecto cuesta $0.

### ¿Dónde veo mi sitio?
**En una URL pública** como: `https://technova.azurewebsites.net`

### ¿Puedo usar mi propio dominio?
**Sí**, pero es opcional. Primero termina con el dominio Azure gratis.

### ¿Qué pasa después?
Puedes:
- Agregar funcionalidades
- Conectar una base de datos
- Automatizar despliegues
- Agregar analytics
- Y mucho más
