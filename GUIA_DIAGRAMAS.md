## 🔧 CÓMO IMPORTAR EN CADA HERRAMIENTA

### ✅ **Draw.io (Web - RECOMENDADO)**

1. Accede a: https://app.diagrams.net/
2. Click en **File** → **Open from Device**
3. Selecciona cualquier archivo `.drawio.xml`
4. ¡Listo! El diagrama se carga automáticamente
5. Puedes editar libremente y guardar en tu Drive

**Ventaja**: Gratuito, sin instalación, sincronización con Drive

---

### ✅ **Draw.io (Desktop)**

1. Descarga: https://github.com/jgraph/drawio-desktop/releases
2. Instala la aplicación
3. Abre: **File** → **Open** → Selecciona `.drawio.xml`
4. Edita y guarda

---

### ✅ **Microsoft Visio**

1. Abre Visio
2. File → Open
3. Selecciona el archivo `.drawio.xml`
4. Visio lo convierte automáticamente
5. Puedes editar y guardar como `.vsdx`

**Nota**: Si necesitas convertir, puedes:
- Guardar en Draw.io como PNG/SVG
- Importar en Visio como imagen
- O usar: https://app.diagrams.net/ y exportar a Visio

---

### ✅ **Lucidchart**

**Opción 1: Import Directo**
1. Accede a: https://www.lucidchart.com/
2. Click **+ New document**
3. **File** → **Open** → Carga `.drawio.xml`
4. Lucidchart lo procesa automáticamente

**Opción 2: Vía Cloud**
1. Sube a Google Drive
2. Right-click → **Open with** → **Lucidchart**

**Nota**: Algunos elementos pueden requerir ajustes de formato

## 💡 EDITAR LOS DIAGRAMAS

Todos los archivos `.drawio.xml` pueden editarse:

1. **Cambiar colores**: Haz clic en elemento → Properties → Fill
2. **Modificar texto**: Doble clic en elemento
3. **Agregar formas**: Arrastra desde el panel izquierdo
4. **Conectar**: Arrastra línea entre elementos

---

## 📤 EXPORTAR DESDE DRAW.IO

Una vez edites, puedes exportar como:

```
File → Export As → 
  ├── PNG (para web/presentaciones)
  ├── SVG (para redimensionamiento)
  ├── PDF (para imprimir)
  ├── VSDX (para Visio)
  └── XML (para guardar cambios)
```
