## 🎯 EJERCICIOS PRÁCTICOS

### Nivel 1: Básico

**Ejercicio 1: Rectángulo Simple**
```
Objetivo: Aprender a crear formas básicas
Tiempo: 5 minutos

Pasos:
1. Abre Draw.io
2. Arrastra 3 rectángulos
3. Agrega texto: "Inicio", "Proceso", "Fin"
4. Conecta con líneas
5. Cambia colores
6. Exporta como PNG
```

**Ejercicio 2: Diagrama de Flujo Simple**
```
Objetivo: Crear decisiones con rombos
Tiempo: 10 minutos

Pasos:
1. Crea círculo (inicio)
2. Crea rombo (decisión)
3. Crea 2 rectángulos (opciones)
4. Crea círculo (fin)
5. Conecta todos
6. Etiqueta flechas
```

**Ejercicio 3: Diagrama de Arquitectura TechNova**
```
Objetivo: Crear diagrama como viste en ejemplos
Tiempo: 20-30 minutos

Pasos:
1. Crea estructura en 4 niveles
2. Agrega colores corporativos
3. Conecta elementos
4. Etiqueta todo
5. Alinea y centra
6. Exporta como PDF
```

**Ejercicio 4: Infraestructura Azure**

## 🔧 SOLUCIÓN DE PROBLEMAS

### Problema: "No puedo conectar las formas"
```
Solución:
1. Asegúrate que ambas formas existan
2. En Draw.io, arrastra línea azul entre formas
3. En Visio, usa herramienta Connector
4. Usa puerto de conexión (punto azul)
```

### Problema: "Los colores se ven diferentes"
```
Solución:
1. Verifica código hex del color (#0078D4)
2. No uses gradientes para imprimir
3. Exporta como PDF en lugar de PNG
4. Prueba en diferentes navegadores
```

### Problema: "El texto es muy pequeño"
```
Solución:
1. Selecciona forma
2. Properties → Font size
3. Mínimo: 10pt para impresión
4. Mínimo: 12pt para web
```

### Problema: "No puedo exportar a VSDX"
```
Solución:
1. En Draw.io: File → Export As → VSDX
2. O guarda como XML y abre en Visio
3. O convierte PNG a VSDX en línea
```
