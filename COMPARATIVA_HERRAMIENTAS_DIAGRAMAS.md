## 🚀 INSTRUCCIONES PASO A PASO

### DRAW.IO (Recomendado para principiantes)

#### Abrir diagrama:
```
1. Ve a: https://app.diagrams.net/
2. Click "File" → "Open"
3. Carga DIAGRAMA_ARQUITECTURA.drawio.xml
4. ¡Listo! Se abre automáticamente
```

#### Editar:
```
- Doble clic en texto para cambiar
- Click en elemento → Properties (abajo)
- Arrastra para mover
- Ctrl+Shift+F para buscar elementos
```

#### Exportar:
```
File → Export As →
  • PNG: Para presentaciones/web
  • PDF: Para imprimir
  • SVG: Para redimensionar sin perder calidad
```

---

### MICROSOFT VISIO

#### Abrir diagrama:
```
1. Abre Microsoft Visio
2. File → Open
3. Busca DIAGRAMA_AZURE_INFRAESTRUCTURA.drawio.xml
4. Visio lo convierte a .VSDX
5. Click "Enable Editing" si te aparece
```

#### Editar:
```
- Más formas disponibles en "Shapes"
- Propiedades en panel derecho
- Más control de líneas y conectores
```

#### Exportar:
```
File → Save As → Formato: VSDX (nativo de Visio)
O → Export → PDF/PNG/SVG
```

---

### LUCIDCHART

#### Abrir diagrama:
```
1. Ve a: https://www.lucidchart.com/
2. Sign up (crea cuenta o usa Google)
3. "+ New document" → "Blank diagram"
4. Click "File" (arriba) → "Import"
5. Carga el archivo .drawio.xml
```

#### Editar:
```
- Panel izquierdo: Shapes Library
- Arrastra formas al canvas
- Conecta con líneas automáticas
- Colaboración en tiempo real disponible
```

#### Compartir:
```
Botón "Share" (arriba derecha)
→ Copia link o invita colaboradores
→ Pueden editar en tiempo real
```

## 💾 GUARDAR Y VERSIONAR

### Estructura de carpetas recomendada:
```
PROYECTO EN NUBE/
├── DIAGRAMAS/
│   ├── v1.0/
│   │   ├── DIAGRAMA_ARQUITECTURA_v1.drawio
│   │   ├── DIAGRAMA_ARQUITECTURA_v1.pdf

│   └── v2.0/
│       └── DIAGRAMA_ARQUITECTURA_v2.drawio
```

### Nomenclatura:
```
DIAGRAMA_[TIPO]_v[VERSION].[FORMATO]

Ejemplo:
DIAGRAMA_ARQUITECTURA_v1.0.drawio
DIAGRAMA_AZURE_INFRAESTRUCTURA_v2.1.pdf
DIAGRAMA_FLUJO_USUARIO_v1.0.png
```

## 🤝 COLABORACIÓN

```
1. Abre draw.io → File → Open from URL
2. Copia link de Drive (compartir)
3. Pega URL en draw.io
4. ¡Edición colaborativa activa!
```

### Con Teams (Visio/Lucidchart):
```
1. Guarda en OneDrive/SharePoint
2. Comparte en Teams
3. Colaboradores pueden abrir y editar
4. Cambios se sincronizan automáticamente
```

## ❓ TROUBLESHOOTING

**Problema**: Archivo no se abre en Draw.io
→ Solución: Intenta como "Open from URL" en lugar de upload local

**Problema**: Colores se ven diferentes en PDF
→ Solución: Exporta como PNG primero, luego convierte a PDF

**Problema**: Líneas de conexión no se alinean
→ Solución: Usa "Arrange" → "Align" para alinear elementos

**Problema**: Archivo muy pesado
→ Solución: Reduce número de shapes o divide en múltiples diagramas
